description: 가벼운 중간 기록 — HISTORY.md에 작업 기록만 남김 (커밋 안 함)
---

세션을 완전히 끝내지 않고, 지금까지 한 작업을 가볍게 기록만 남긴다. (커밋은 하지 않음)

지금까지 이 세션에서 한 작업을 아래 형식으로 `docs/HISTORY.md` **맨 위**에 추가 (기존 내용은 아래로 밀기):

```
## YYYY-MM-DD HH:MM (세션 요약 제목)

**한 일**
- (이번 세션에서 실제로 수정/생성한 것. 파일명 포함)

**수정한 파일**
- 파일경로 — 무엇을 바꿨는지 한 줄

**현재 상태**
- (지금 어디까지 됐는지)

**다음 할 일**
- (이어서 할 것)

**주의/메모**
- (막힌 점, 검증 안 된 것 등. 없으면 생략)

---
```

## 규칙
- 날짜·시간은 실제 현재 시각.
- 한국어, 개조식, 간결하게. 실제로 한 것만.
- **git 커밋은 하지 않는다** (기록만). 커밋이 필요하면 사용자가 `/end`를 쓰거나 직접 지시.
- 다 적은 뒤: "HISTORY.md에 중간 기록 완료 (커밋 안 함). 계속 작업하거나, 마무리하려면 /end" 한 줄 안내.
